## 一、wsl启动失败问题

当wsl启动报错显示：`参考的对象类型不支持尝试的操作`

### 临时方法

管理员身份执行:

```bash
netsh winsock reset
```

### 永久方法

下载工具：`NoLsp.exe` 

[下载地址](https://richard-li.lanzouv.com/ibT8a09kvp9a)

管理员模式下执行：

```bash
.\NoLsp.exe C:\windows\system32\wsl.exe
```
